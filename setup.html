⭐ Chore Stars ⭐
Set up your family's chore tracker in 3 steps
Self-hosted  ·  No account needed  ·  Runs on your computer
Step 1 of 3
🐋 Install Docker Desktop

Docker is a free app that runs the whole stack for you — no fiddling with databases or Node.js. Download it for your computer:

🪟 Windows 🍎 Mac (Apple chip) 🍎 Mac (Intel) 🐧 Linux
Already have Docker? Skip straight to step 2. Just make sure Docker Desktop is open and running before step 3.
Step 2 of 3
📦 Download the code

Go to github.com/jessekward-prog/chore-stars and click the green Code button, then Download ZIP.

Unzip it somewhere easy to find — your Desktop or Documents folder works great.

Have Git installed? You can also run:
git clone https://github.com/jessekward-prog/chore-stars.git
Step 3 of 3
🚀 Start it up

Open a terminal (Windows: search "Command Prompt" or "PowerShell" · Mac: search "Terminal") and navigate into the folder you unzipped:

cd path/to/chore-stars

Then run:

docker compose up --build
The first run takes 2–3 minutes to download and build everything. You'll see a lot of text scroll past — that's normal. When it says Chore Stars running at http://localhost:3001 you're ready.

Then open your browser and go to:

http://localhost:3001
🔒 First time setup
1️⃣
Tap the 🔒 lock icon in the top-right corner of the chore screen.
2️⃣
The default password is stars123 — enter it and tap Unlock.
3️⃣
Go to the Security tab and change the password to something only you know.
4️⃣
Go to the Kids tab to rename the kids, add more, or upload their photos.
5️⃣
Go to the Prizes tab and fill in the wheel slots — Movie Night, Ice Cream Trip, Stay Up Late… whatever motivates yours.
📱 Using it on phones and tablets
Access from any device on your home wifi
The app runs on the computer you installed it on. Other devices on the same wifi can reach it using that computer's local IP address instead of localhost.

Find your computer's local IP:
 • Windows: Open Command Prompt → type ipconfig → look for IPv4 Address
 • Mac: System Settings → Network → your connection → IP Address

Then on your phone, go to http://192.168.x.x:3001 (using the actual IP you found). Add it to your home screen for an app-like experience.
🔁 Keeping it running
💡
The app keeps running as long as Docker Desktop is open. If you restart your computer, just open Docker Desktop and the container will start automatically — or run docker compose up again from the project folder.
💾
Your data is safe. Everything (kids, chores, history) is stored in a Docker volume — it survives restarts, updates, and rebuilds.
