⭐ Chore Stars ⭐
Set up your family's chore tracker in 3 steps
Self-hosted  ·  No account needed  ·  Runs on your computer
Step 1 of 3
🐋 Install Docker

Docker runs the whole stack for you — no fiddling with databases or Node.js. Pick your OS:

🪟 Windows — open PowerShell and run:
winget install Docker.DockerDesktop
winget is built into Windows 10/11. Or download the installer instead.
🍎 Mac — open Terminal and run:
brew install --cask docker
Requires Homebrew. Or download: Apple chip · Intel.
🐧 Linux — open a terminal and run:
curl -fsSL https://get.docker.com | sh
Official Docker install script — works on Ubuntu, Debian, Fedora, and most distros.
Already have Docker? Skip to step 2. Make sure Docker Desktop is open and running before step 3.
Step 2 of 3
📦 Download the code

Click this link to open the project page on GitHub:

🐙 github.com/jessekward-prog/chore-stars

Once you're on that page:

  1. Look for the big green "Code" button near the top right of the file list
  2. Click it — a small dropdown appears
  3. Click "Download ZIP" at the bottom of that dropdown
Once downloaded, unzip the file somewhere easy to find — your Desktop or Documents folder works great. You should end up with a folder called chore-stars-main.
Have Git installed? You can also run:
git clone https://github.com/jessekward-prog/chore-stars.git
Step 3 of 3
🚀 Start it up

You need to open a terminal inside the folder you unzipped. The easiest way depends on your OS — no typing the path needed:

🪟 Windows
Open the unzipped folder in File Explorer. Click the address bar at the top (where it shows the folder path) — it'll highlight the path. Type cmd and press Enter. A Command Prompt opens already inside that folder.
🍎 Mac
Right-click (or Control+click) the unzipped folder in Finder and choose "New Terminal at Folder". If you don't see that option, go to System Settings → Privacy & Security → Developer Tools and enable Terminal.
🐧 Linux
Right-click inside the unzipped folder in your file manager and choose "Open Terminal Here".

Once the terminal is open in the right folder, run:

docker compose up --build
The first run takes 2–3 minutes to download and build everything. You'll see a lot of text scroll past — that's normal. When you see Chore Stars running in the terminal, you're ready.

Then open your browser and go to:

http://localhost:3013
✨ First time setup
🧙
The first time you open the app, a setup wizard runs automatically. It walks you through adding your kids, choosing their daily chores, naming the prize wheel slots, and setting a parent password — all before anything else appears.
🔒
After setup, tap the lock icon anytime to get back into the parent settings — rename kids, change chores, update prizes, or change your password.
📱 Using it on phones & tablets
Access from any device on your home wifi
localhost only works on the computer running Docker. Phones, tablets, and other computers on the same wifi need to use your computer's local IP address instead.

Step 1 — find your computer's local IP:
 • Windows: Open Command Prompt → type ipconfig → look for IPv4 Address (usually starts with 192.168.)
 • Mac: System Settings → Network → your wifi → IP Address
 • Linux: Open a terminal → type hostname -I

Step 2 — open on your tablet or phone:
Type that IP address into the browser with :3013 on the end — for example:

http://192.168.1.42:3013

Step 3 — add to your home screen (optional but great):
 • iPhone/iPad: Tap the Share button → "Add to Home Screen"
 • Android: Tap the browser menu → "Add to Home Screen"

This makes it launch fullscreen like a real app — no browser bar.
🔁 Keeping it running
💡
The app keeps running as long as Docker Desktop is open. If you restart your computer, just open Docker Desktop and the container will start automatically — or run docker compose up again from the project folder.
💾
Your data is safe. Everything (kids, chores, history) is stored in a Docker volume — it survives restarts, updates, and rebuilds.
☢️ Remove everything

Want to completely remove Chore Stars from your computer? Run this from inside the project folder. It stops the app, deletes all containers, removes the built image, and permanently deletes all your data (kids, chores, history — everything).

docker compose down -v --rmi all
🗑️ -v — deletes the database volume (all your data)
🗑️ --rmi all — removes the built Docker image
Then just delete the chore-stars folder and Docker has no trace of it.
